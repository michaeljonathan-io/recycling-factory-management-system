# Recycling Factory Management System

## Overview
This project is a web-based operational management system developed for CV. Berkat Anugrah, a plastic recycling company.

The system helps manage:
- Raw material purchasing
- Production processes
- Plastic pellet sales
- Inventory management
- Operational reporting

This project was developed as my undergraduate thesis project.

---

## Features

### Super Admin
- User management
- Plastic material management
- Plastic pellet management
- Report generation
- Price management

### Purchasing Division
- Purchase transaction management
- Supplier management
- Automatic purchase calculation

### Production Division
- Production recording
- Stock monitoring
- Automatic inventory adjustment

### Sales Division
- Sales transaction management
- Customer management
- Automatic stock reduction
